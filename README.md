# tui-bible
Read the Bible in any translation straight from the terminal.

![Preview](preview.png)

## Installation

Using pipx:
```shell
$ python3 -m pipx install git+https://github.com/noahsc0tt/tui-bible@master
```
Using pip:
```
$ python3 -m pip install git+https://github.com/noahsc0tt/tui-bible@master
```

## Usage 
```shell
$ bible
```
Use the arrow keys to navigate and make selections.
